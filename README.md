# Akilli-is-bulma-asistani
 Python ve Flask ile geliştirilen bu proje, CV'nizi analiz ederek size en uygun iş ilanlarını internetten otomatik toplar. Google Gemini yapay zekasıyla aday-ilan uyumluluğunu puanlar, teknik ve deneyim bazlı analizler yaparak kişiye özel kariyer tavsiyeleri sunar. İş arama sürecini dijitalleştirip hızlandıran akıllı bir asistandır.
